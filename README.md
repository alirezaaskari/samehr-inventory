# Samehr Inventory 1.0.0

پکیج کامل مدیریت موجودی سامهر شامل:

- `wordpress-plugin/samehr-inventory-api`: افزونه وردپرس و ووکامرس
- `android-app`: سورس اپلیکیشن اندروید (Kotlin)

## نصب افزونه

1. پوشه `samehr-inventory-api` را جداگانه ZIP کنید.
2. در وردپرس به افزونه‌ها ← افزودن ← بارگذاری افزونه بروید.
3. ZIP را نصب و فعال کنید. ووکامرس باید فعال باشد.
4. از ووکامرس ← Samehr Inventory حد هشدار کمبود را تعیین کنید.
5. برای هر کاربر از کاربران ← شناسنامه، یک Application Password بسازید.

## نقش‌ها و دسترسی

- مدیر و Shop Manager امکان مشاهده و تغییر موجودی دارند.
- کاربران فقط‌خواندنی امکان مشاهده دارند، اما تغییر موجودی نیازمند `edit_products` یا قابلیت `samehr_manage_inventory` است.
- برای نقش سفارشی «مدیر موجودی سامهر»، قابلیت `samehr_manage_inventory` را فعال کنید.

## اجرای اپ

1. پوشه `android-app` را با Android Studio باز کنید.
2. صبر کنید Gradle Sync تمام شود.
3. اپ را روی گوشی یا شبیه‌ساز اجرا کنید.
4. آدرس `https://samehrstore.ir`، نام کاربری وردپرس و Application Password را وارد کنید.

## نکات امنیتی

- سایت حتماً باید HTTPS داشته باشد.
- رمز اصلی وردپرس را در اپ وارد نکنید؛ فقط Application Password.
- در نسخه 1.0.0 رمز برنامه صرفاً در حافظه اجرای اپ نگه داشته می‌شود و روی دستگاه ذخیره نمی‌شود.
