# راهنمای آپلود در GitHub

برای مخزن `alirezaaskari/samehr-inventory`، محتویات داخل پوشه `samehr-inventory-package` را در شاخه `main` آپلود کنید؛ خود پوشه مادر را به‌عنوان یک لایه اضافه آپلود نکنید.

ساختار نهایی مخزن باید چنین باشد:

```text
README.md
UPLOAD_TO_GITHUB.md
android-app/
wordpress-plugin/
samehr-inventory-api-v1.0.0.zip
```

فایل `samehr-inventory-api-v1.0.0.zip` مخصوص نصب مستقیم از پیشخوان وردپرس است. فایل ZIP کامل پروژه را داخل مخزن قرار ندهید؛ محتویات آن را استخراج و آپلود کنید.
