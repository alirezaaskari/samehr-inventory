Secure inventory endpoints for the Samehr Android application.

== Installation ==
1. Upload the samehr-inventory-api folder in Plugins > Add New > Upload Plugin.
2. Activate WooCommerce and this plugin.
3. Set the low-stock threshold under WooCommerce > Samehr Inventory.
4. For each app user, create a WordPress Application Password in Users > Profile.

== Security ==
Use HTTPS. Never use the main WordPress password in the mobile app.
